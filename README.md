# SerchBook-GoogleApi

Приложение поиска книг по googleApi и сортировкой поиска

Технологии: React/Redux/Redux-thunk/React-hooks/Router.v6/PropTypes/css-media/Material-UI/GoogleAPI

Для запуска используйте: 

```
npm i

npm start
```
